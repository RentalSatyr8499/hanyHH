# Third-party assets and licenses

This file lists sources and license terms for creative assets used in **Accessible Routing**. Keep it with the project for submissions, demos, and any store release. **Fonts and stock terms can restrict commercial use** — verify before publishing beyond a school/hackathon context.

---

## Wood photographs / textures

### Unsplash — dark wood grain

- **Work:** [Close-up of dark wood grain texture](https://unsplash.com/photos/close-up-of-dark-wood-grain-texture-KKKHaEA4coc)
- **Uploader:** [The Cleveland Museum of Art](https://unsplash.com/@clevelandart) (@clevelandart)
- **License:** Dual: [Public Domain](https://help.unsplash.com/en/articles/5754412-dual-usage) & [Unsplash License](https://unsplash.com/license) (see photo page for details)
- **Bundled in app:** Used as part of the wild-west theme (e.g. `assets/theme/wild_west/wood_background.jpg` and/or `wood_plank.jpg`, depending on your export)

### Freepik — wood plank / texture

- **Work:** [Three kinds wood](https://www.freepik.com/free-photo/three-kinds-wood_19138518.htm) (Freepik stock photo)
- **License:** Follow [Freepik’s license](https://www.freepik.com/legal/terms-of-use) for the account type you used (free with attribution is common; **always include the attribution line Freepik requires** on the page for that asset — copy it from the download page if shown)
- **Bundled in app:** May be used for `wood_background.jpg` and/or `wood_plank.jpg` (resize/compress as needed for the APK)

> If you use **both** Unsplash and Freepik in the same theme, list which file came from which source in your submission appendix or here.

## Fonts (DaFont)

### Carnivalee Freakshow — button / display text

- **Source:** [Carnivalee Freakshow on DaFont](https://www.dafont.com/carnivalee-freakshow.font)
- **Author:** Christopher Hansen  
- **Bundled file:** `assets/theme/wild_west/Carnevalee_Freakshow.ttf` (family: `CarnevaleeFreakshow` in `pubspec.yaml`)
- **License (per DaFont listing):** “Free for personal use” — confirm [DaFont](https://www.dafont.com/) and any enclosed readme for your use case; **commercial** use may require a separate license from the author.

### Confetti Western — labels / headings

- **Source:** [Confetti Western on DaFont](https://www.dafont.com/confetti-western.font?text=Source+Destination)
- **Author:** Chequered Ink  
- **Bundled file:** `assets/theme/wild_west/Confetti_Western.otf` (family: `ConfettiWestern` in `pubspec.yaml`)
- **License:** Author notes **free for non-commercial (personal) use**; **commercial** use requires a [purchased license](http://chequered.ink/font-license/) — see enclosed `FSLA_NonCommercial_License.html` if distributed with the download.
